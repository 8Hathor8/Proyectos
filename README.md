# Sobre mí

¡Hola! Soy **Ángel**, estudiante de **ASIR** (Administración de Sistemas Informáticos en Red). Me apasiona el mundo de la informática, la administración de redes y la gestión de servidores. Actualmente, estoy adquiriendo habilidades para configurar, gestionar y mantener infraestructuras IT tanto en el ámbito local como en la nube.

## Habilidades
- **Administración de sistemas Linux y Windows**
- **Redes (TCP/IP, subredes, enrutamiento)**
- **Virtualización (VMware, VirtualBox)**
- **Automatización (Bash, PowerShell)**
- **Seguridad informática y firewalling**
- **Manejo de servidores web (Apache, Nginx)**

## Proyectos
- **Gestión de redes empresariales**: Implementación de una red corporativa en un entorno virtualizado, con servidores DNS, DHCP y un servidor web.
- **Configuración de servidores Linux**: Instalación y administración de servidores en Debian/Ubuntu para distintos servicios como correo electrónico, bases de datos y páginas web.
